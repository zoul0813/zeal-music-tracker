## Controls

Refer to on-screen controls

* S - Save File
* L - Load File
* Space - Play/Stop
* Esc - Quit
* H - Inline Help
* P - Pattern View
* A - Arrangement View

### All Views

* Up/Down - Next Step
* Home/End - First/Last Step
* Left/Right - Adjust Step
* PgUp/PgDown - Adjust Step Plus
* Ins - Duplicate Last Step
* Del - Delete Step
* Tab - Next Cell

### Arrangement View

### Pattern View

* 1-4 - Voice 1-4
* [/] - Prev/Next Pattern
* N - New Pattern

### Arrangement FX

* [X] 0x00-0x1F - goto arrangement
* [X] 0x20-0x3F - tempo ((value - 0x20 + 1 * 4))
* [ ] 0xD0-0xDC - transpose by (value - 0xD0) steps

### Pattern FX

* [ ] 0x00 - note off
* [ ] 0x01 - note on
* [X] 0x10 - Square
* [X] 0x11 - Triangle
* [X] 0x12 - Saw
* [X] 0x13 - Noise
* [X] 0xC0-0xC8 - Countdown (value - 0xC0)
* [X] 0xD0-0xEF - Goto Step (value - 0xD0)
* [X] 0xF0-0xF8 - Volume (0, 12.5, 25, 37.5, 50, 62.5, 75, 87.5, 100)
