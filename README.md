## Controls

Refer to on-screen controls

* S - Save File
* L - Load File
* Space - Play/Stop
* Esc - Quit
* H - Inline Help
* P - Pattern View
* A - Arrangement View

### All Views

* Up/Down - Next Step
* Home/End - First/Last Step
* Left/Right - Adjust Step
* PgUp/PgDown - Adjust Step Plus
* Ins - Duplicate Last Step
* Del - Delete Step
* Tab - Next Cell

### Pattern View

* 1-4 - Voice 1-4
* [/] - Prev/Next Pattern
* N - New Pattern
